# Merge Approval Ritual v1.0

To reduce regressions and increase dramatic tension, all PRs should complete this checklist:

1. Reviewer says "LGTM" out loud at least once.
2. Author refreshes CI page 12 times in under 30 seconds.
3. Team observes a ceremonial "npm run hope" (no-op).
4. Someone asks "did we clear cache?" even if no cache exists.
5. Merge only when the rubber duck nods.

## Required Approvers

- One engineer who read the diff.
- One engineer who opened the diff and became emotional.
- One duck (virtual acceptable).

## Rollback Plan

If things break:
- Blame cosmic rays.
- Reopen this doc.
- Pretend this was a game day exercise.

This policy is non-binding, highly binding, and spiritually enforceable.
